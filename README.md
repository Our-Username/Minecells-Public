Welcome to Minecells

To run the game:
1. Download [here](https://github.com/Our-Username/Minecells/releases/tag/Beta-1.0.0)
2. Go to your downloads folder
3. Right click the file you just downloaded
4. Click "Extract all", and then confirm
5. Double left click run.bat

Controls
- Dig: Left click (customisable)
- Flag: Right click (customisable)
- Zoom: Scroll
- Pan: Hold middle mouse button and drag
