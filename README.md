Welcome to Minecells

To run the game:
1. Download [here](https://github.com/Our-Username/Minecells-Public/releases/tag/Beta-v1.0.4)
2. Click on Source code (zip)
3. Go to your downloads folder
4. Right click the file "Minecells-Public-Beta-v1.0.4.zip"
5. Click "Extract all", and then confirm
6. Open the folder "Minecells-Public-Beta-v1.0.4"
7. Double click run.bat
8. If a security popup comes up, click run

Controls
- Dig: Left click (customisable)
- Flag: Right click (customisable)
- Zoom: Scroll
- Pan: Hold middle mouse button and drag
