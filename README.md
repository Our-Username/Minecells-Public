Welcome to Minecells

To run the game:
1. Download [here](https://github.com/Our-Username/Minecells-Public/releases/tag/Beta-1.0.0)
2. CLick on "Source code (zip)"
3. Go to your downloads folder
4. Right click the file you just downloaded (should be called "Minecells-Public-Beta-1.0.0.zip)
5. Click "Extract all", and then confirm
6. Double click the folder called "Minecells-Public-Beta-1.0.0"
7. Double click the file "run.bat"
8. If a security popup appears, click run

Controls
- Dig: Left click (customisable)
- Flag: Right click (customisable)
- Zoom: Scroll
- Pan: Hold middle mouse button and drag
